Stanfy Open Source

We use open source to build our own products and provide services. And here are some projects we develop internally and release for community. https://github.com/stanfy

gson-xml

Java library for XML deserialization.

DyCI

Dynamic code injection for iOS projects - powerful tool that accelerates apps development in Xcode and AppCode environments.

enroscar

A set of libraries we actively use to develop for Android platform.

mattock

JUni4 tests runner for Android.

gae-go-testing

Go lang package for writing tests for AppEngine platform.

maven-repo

Maven repository of commonly used libraries.
